# Dean's Blog
### Individual Project
Blog website
### Stacks
* Front-end: React, Redux
* Back-end: Node.Js, Express
* Database: MongoDB
* Testing: Jest
* Containerization: Docker, Docker-Compose
* CI/CD: Travis CI
* Web Hosting: AWS(Elastic Beanstalk)

### Features
* Json Web Token
* OAuth - Reset Password(Gmail)
* Creat, Read, Update, Delete functions.
